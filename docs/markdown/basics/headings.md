## ATX headings

ATX headings need not be separated from surrounding content by blank lines, and they can interrupt paragraphs:

The following is a h3 surrouned by lines:
****
### foo
****
 
The following is a h3 interrupting 2 paragraphs:

Foo bar
### baz
Bar foo
